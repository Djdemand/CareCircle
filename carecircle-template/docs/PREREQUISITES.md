# Prerequisites

Before setting up CareCircle, please ensure you have the following ready.

## üíª Software Requirements

### 1. Node.js (Runtime Environment)
*   **Version:** v18.0.0 or higher (LTS recommended)
*   **Why:** Required to run the build tools and setup scripts.
*   **Download:** [https://nodejs.org/](https://nodejs.org/)
*   **Verify:** Open terminal/command prompt and run `node -v`

### 2. Git (Version Control) - Optional
*   **Why:** Recommended for managing your code and easier deployments.
*   **Download:** [https://git-scm.com/downloads](https://git-scm.com/downloads)

### 3. Code Editor - Optional
*   **Why:** To view and edit the code.
*   **Recommendation:** [Visual Studio Code](https://code.visualstudio.com/)

*   **Why:** Hosts the web application so it's accessible online.
*   **Cost:** Free tier is sufficient.
*   **Sign Up:** [https://www.netlify.com/](https://www.netlify.com/)

### 3. Expo (Mobile App) - Optional
*   **Why:** Required only if you want to build and publish the mobile app.
*   **Cost:** Free tier available.
*   **Sign Up:** [https://expo.dev/](https://expo.dev/)
